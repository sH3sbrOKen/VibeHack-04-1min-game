# 文案需求：伪人乱入结局（2条）

**需求方**：主程序
**优先级**：🔴 路演前必须
**背景**：SPACETIME_RADIO 故事线第15秒会触发「伪人乱入」（A·M 发来一条消息），玩家操作与否决定走哪条结局分支。

---

## 触发说明

| 结局 | 触发条件 |
|------|---------|
| 伪人结局1 | 乱入10秒内**不操作**任何按钮 |
| 伪人结局2 | 乱入10秒内**点了任意按钮** |

两条结局都在**正常结算页**显示（不打断游戏），优先级高于其他 SPACETIME_RADIO 结局。

---

## 乱入消息内容（参考用）

```
发信人：A·M
内容：我找到你了。
      你一直在这个频率。
      回复我。
```

---

## 交付文案

### 伪人结局1（不操作，安全）

> 玩家没有回应。正确做法。后怕感。

```
标题：没有回应
图标：🔇
文案：你没有回复。
      那个频率重新安静下来。
      它在等了六秒之后，切断了。
      （如果你回了，它会收到。它知道在哪里找你。）
```

### 伪人结局2（有操作，触发）

> 玩家回复了。触发警惕紧张感。

```
标题：已收到
图标：🔴
文案：你回复了。
      A·M 收到了你的信号。
      它现在也知道这条线路是活的。
      （它知道在哪里找你了。）
```

---

## 文案风格提醒（COPY-STYLE 摘要）

- 冷淡陈述，不评价，不煽情
- 无感叹号，无「居然」「竟然」等情绪词
- 括号里放一句反差极大的日常观察
- 最后停在奇怪的地方

**反面示例**：
```
❌ 你竟然没有回复！太聪明了！
❌ 你成功识破了伪人的伪装，避免了一场灾难。
```
