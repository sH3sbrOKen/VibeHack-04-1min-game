# STORYLINE-SCAM_TARGET

## A. 基本信息

- **ID**: SCAM_TARGET
- **名称**: 被盯上一分钟
- **开场旁白**:
  ```
  你的号码进了一个名单。
  名单上的每个人都很忙。
  （他们专门盯着你。）
  ```
- **进入条件**: 主线 对SPAM类型的错误回应占比 ≥ 所有类型中最高
- **脱离条件**:
  - 60秒内完全不点击任何按钮 → 结局「消失术」，脱离
  - 所有点击均为UNSUBSCRIBE → 结局「较真」，进入世界线（待定）
