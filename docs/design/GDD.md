## 一、游戏概述

**背景故事**：主角起床，发现充电器昨晚没插紧，手机只剩1%电。  
**目标**：60秒内用四个预设短语回复99+条来自不同人的消息。  
**核心张力**：时间压力 × 判断压力 × 隐藏后果

### 体验弧线（关键设计意图）

```
99+条消息，不可能全部回完
  → 玩家意识到来不及，开始自暴自弃乱回
  → 答错触发隐藏数值，屏幕冒出无厘头的数字
  → 一局结束，发现"失败"竟然解锁了新故事
  → 反转：原来乱回才是正确的玩法
```

**99+是真实数字**：消息由手写卡牌 + 程序化生成补足，角标从99+开始递减。通关（回完全部）几乎不可能——这是故意的。恭喜弹窗是一个极度稀有的彩蛋，不是常规体验路径。

## 二、核心循环（局内 Inner Loop）

```
消息弹出 → 玩家判断类型 → 点击四角之一 → 
  正确：消息消失，继续
  错误：消息消失 + 对应数值变化（玩家可能没注意到）
→ 60秒倒计时结束 → 展示本局数值结果
```

### 消息弹出节奏

| 时间段 | 特点 |
|------|-----|
| 0~15秒 | 难度1消息为主，节奏稍慢，让玩家熟悉四个按钮 |
| 15~40秒 | 难度1+2混合，加入陷阱卡 |
| 40~60秒 | 难度2+3为主，消息间隔缩短（100ms），节奏加速 |

### 四角回复按钮

| 位置 | 文字 | 对应 ReplyType |
|-----|-----|-------------|
| 左上 | 等等 | WAIT |
| 右上 | 收到 | RECEIVED |
| 左下 | 退订 | UNSUBSCRIBE |
| 右下 | 好的亲 | LOVE |

## 三、数值系统

### 设计哲学

数值是这个游戏最重要的叙事装置，不是辅助系统。

**无厘头是特性**：数值名称和出现方式要让玩家产生"这是什么鬼"的反应。它们不该像正经游戏的属性面板，而更像手机弹出的莫名其妙的通知——跟这个游戏的"荒诞日常"主题一脉相承。

**只在变化时显示**：数值默认完全隐藏。只有答错导致某个数值变化时，该数值才首次浮现。玩家全程答对就永远看不到这些数值——但那几乎不可能。

**数量不设上限**：当前6个，未来可以随内容扩展增加。越多越好——荒诞感来自"怎么又冒出一个"的累积效应。新增数值时只需要：名称够无厘头 + 对应至少一张信息卡触发 + 绑定一条支线故事。

**长循环反转**：玩家第一局觉得这些数值莫名其妙，几局后发现每个数值都对应一条支线故事的入口——"原来这些无厘头的东西是有用的"。这个认知反转是 meta 循环的核心驱动力。

| 数值名 | 方向 | 初始值 | 触发阈值 | 满值文案 |
|------|-----|------|--------|--------|
| 外卖丢失率 | 越高越坏 | 0 | ≥ 3 | 「你的外卖已被退回，骑手也很委屈」 |
| 老板耐心值 | 越低越坏 | 100 | ≤ 0 | 「老板已将你移出工作群」 |
| 个人信息泄露指数 | 越高越坏 | 0 | ≥ 3 | 「恭喜！你的号码已进入全国骚扰名单」 |
| 亲密度 | 越低越坏 | 100 | ≤ 0 | 「对象已读不回」 |
| 骚扰订阅量 | 越高越坏 | 0 | ≥ 5 | 「您已成功订阅来自全国各地的关心」 |
| 意外扣费金额 | 越高越坏 | 0 | ≥ 200 | 「本月超支¥888，余额已归零」 |

## 五、局外循环（Outer Loop / Meta Loop）

**「一分钟」是入口，不是游戏本体。** 主线建立操作习惯，真正的游戏从结算页开始。

```
第一局 → 结算数值 → 解锁平行时间线入口 →
进入时间线（独立世界观，部分有颠覆性机制） →
时间线有独立结算 + 可能解锁更深层支线
```

第一局失误率越高，解锁入口越多——**失败是探索的奖励**。

**终极目标：100个结局。** 每条时间线短小精悍，整体碎片化拼出一个更大的世界。支线不局限于"换一批发信人"，部分支线机制完全颠覆：有的要求完全不操作，有的翻转视角，有的有时序依赖。

完整设计见 [META-LOOP.md](META-LOOP.md)。

## 六、支线关卡设计

### STORYLINE_RIDER — 骑手一分钟
- **解锁条件**：外卖丢失率 ≥ 3
- **开场旁白**：单量第38单。导航又在转圈。楼道里没有灯。回复那些等着饭的人——你只有四句话。
- **发信人池**：催单用户、平台客服、餐厅、同事骑手
- **难度**：2

### STORYLINE_CRISIS — 职场危机一分钟
- **解锁条件**：老板耐心值 ≤ 0
- **开场旁白**：会议室里，五个人都在看你。你的手机也快没电了。只是这次，你是老板。
- **发信人池**：下属、客户、HR、竞争对手
- **难度**：2

### STORYLINE_SALES — 电话销售一分钟
- **解锁条件**：个人信息泄露指数 ≥ 3
- **开场旁白**：名单上有200个号码。每一个接通的背后，都有一个正在忙碌的人。你只有一分钟的话术，和四种开口方式。
- **发信人池**：接听用户（分类：有意向/无意向/挂断/骂人）
- **难度**：3

### STORYLINE_BREAKUP — 分手现场一分钟
- **解锁条件**：亲密度 ≤ 0
- **开场旁白**：你知道你说错了什么。你不知道怎么说对。还有六十秒，消息在一条条进来。
- **发信人池**：对象（全部带情绪，无标准答案感）
- **难度**：3

### STORYLINE_OPERATOR — 骚扰接线员一分钟
- **解锁条件**：骚扰订阅量 ≥ 5
- **开场旁白**：这份工作的考核指标只有一个：判断对方是真人还是机器。前三十秒你以为你做到了。
- **发信人池**：来电方（真人/机器人/真人装机器人/机器人装真人）
- **难度**：2

### STORYLINE_SUPPORT — 客服投诉一分钟
- **解锁条件**：意外扣费金额 ≥ 200
- **开场旁白**：客户很生气。你的权限只够说四种话。系统里没有退款按钮。
- **发信人池**：愤怒用户（诉求各异）
- **难度**：3

## 七、文案包装

### 游戏名备选

| 名称 | 风格 | 说明 |
|-----|-----|-----|
| 「1%」 | 简洁数字感 | 直接说出绝境 |
| 「99+」 | 场景感 | 强调消息量压迫 |
| 「今天不想回消息」 | 荒诞反向（推荐） | 偏偏你必须回，最有梗 |

### 主页一句话
> 手机1%电，99+消息，四个字救不了所有人。

### 结算分享卡片
> 我玩了一局「今天不想回消息」  
> 外卖丢了{N}单 / 老板耐心剩{N}% / 泄露了{N}次个人信息  
> 你能比我强吗？

### 黑客松30秒Pitch
> 你有没有经历过——手机1%电，消息99+，但每条都不能不回。  
> 这个游戏就是这一分钟。  
> 四个角，四种回复，你要快速判断老板的消息、外卖的消息、诈骗短信、对象。  
> 回错了，荒诞的数值开始飙升。  
> 回对了，你以为你赢了——直到你发现，第一局的失误，已经开启了另一个故事。
