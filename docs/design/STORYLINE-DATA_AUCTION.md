# STORYLINE-DATA_AUCTION

## A. 基本信息

- **ID**: DATA_AUCTION
- **名称**: 名单拍卖一分钟
- **开场旁白**:
  ```
  你的号码正在被竞价。
  七家公司，实时出价。
  （最高出价是四毛三。）
  ```
- **进入条件**: 由世界线 SCAM_CHAIN 指定（SCAM_TARGET 全部退订脱离）
- **脱离条件**: 60秒内完全不操作 → 结局「流拍」，脱离
