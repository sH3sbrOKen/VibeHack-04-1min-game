# STORYLINE-SPACETIME_RADIO — 超时空电波一分钟

## A. 基本信息

- **ID**: SPACETIME_RADIO
- **名称**: 超时空电波一分钟
- **开场旁白**:
  ```
  1986年。Vorkuta-5。
  一座在地图上不存在的城市。
  有人爬上了市政塔，向宇宙发出信号。
  （她不知道有人在听。）
  ```
- **进入条件**: 由世界线 SPACETIME_CHAIN 指定（乱入触发后玩家点「收到」）
- **脱离条件**: 本线 SIGNAL_STRENGTH ≥ 80（持续回复收到维持信号强度，联络成功）

---
